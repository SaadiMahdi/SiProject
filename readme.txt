Présentation des packages et librairies :
Pour le backend:
Express : Un framework pour construire des applications web avec Node.js.
version : "^4.18.2"


Mongoose: Un ODM pour MongoDB pour faciliter l'interaction avec les bases de données.
version : "^6.8.1"


Morgan : Un middleware pour logger les requêtes et les réponses
version : "^1.10.0"


Cors : une fonctionnalité de sécurité pour contrôler les domaines autorisés à accéder aux ressources de l'application
version : "^2.8.5"


Dotenv : un module qui charge les variables d'environnement à partir d'un fichier .env
version : "^16.0.3"




Pour le frontend : 
Vue: un framework javascript pour construire des interfaces utilisateurs.
version : "^2.6.14"


Vuetify : un framework de composants Material Design pour Vue.js.
version :  "^2.6.0"


Vue-router : un module officiel pour gérer les routes côté client dans une application Vue.js.
version : "^3.5.1"


FontAwesome : une bibliothèque d'icônes populaire qui fournit des icônes vectorielles évolutives qui peuvent être personnalisées (taille, couleur, ombre, etc.) en utilisant CSS.
version : "^6.2.1"


Chart.js : une bibliothèque JavaScript pour créer des graphiques et des visualisations de données sur le navigateur.


Pour pouvoir utiliser l’application : 
Accéder au dossier du front end  en utilisant les commandes 
-cd frontend 
-cd concept 
puis pour installer les packages utilisez la commande : 
-npm i 
et re faire la même chose avec le dossier backend 
-cd backend 
puis pour installer les packages utilisez la commande : 
-npm i
