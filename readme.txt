Mongoose: Un ODM (mapping objet-document) pour MongoDB qui facilite l'interaction avec les bases de données MongoDB.
Il permet de définir la structure des données, de valider les données et de gérer les relations entre les collections.

Morgan: Un middleware qui log les requêtes et les réponses dans la console ou un fichier. 
Il vous aide à suivre les requêtes envoyées à votre application et peut être utile pour déboguer ou surveiller l'application.

Cors: CORS (Cross-Origin Resource Sharing) est une fonctionnalité de sécurité qui permet à une application 
web de contrôler les autres domaines qui peuvent accéder à ses ressources.
Il vous permet de spécifier les origines autorisées à faire des requêtes à votre application.

Dotenv: Un module sans dépendance qui charge les variables d'environnement à partir d'un fichier .env. 
Il vous permet de stocker les paramètres de configuration spécifiques à un environnement dans un fichier séparé,
cela aide à conserver les informations sensibles telles que les identifiants à l'écart de votre système de contrôle de version.
